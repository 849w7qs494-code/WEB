## 📌 Описание задания

1. Установить Composer и библиотеку GuzzleHttp
2. Создать класс ApiClient для работы с API hh.ru/areas
3. Создать класс UserInfo для получения информации о пользователе
4. Интегрировать API в форму регистрации студента (список стран)
5. Сохранять данные в сессию, файл и куки
6. Отображать полученные данные из API на главной странице

### Результат доступен по адресу:
**Главная страница:** http://localhost:8080/index.php 

**Форма регистрации:** http://localhost:8080/form.html

**Все сохранённые данные:** http://localhost:8080/view.php

## ⚙️ Как запустить проект

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/849w7qs494-code/WEB.git
   cd "WEB/lab 4"
2. Запустить контейнеры:
```bash
docker-compose up -d --build
```
3. Открыть в браузере:

```http://localhost:8080/index.php``` - PHP информация

```http://localhost:8080/form.html``` - форма регистрации

```http://localhost:8080/view.php``` - все записи

📂 Содержимое проекта

```www/form.html``` - форма регистрации студента

```www/index.php``` - главная страница

```www/ApiClient.php``` - класс для работы с API hh.ru

```www/UserInfo.php``` - класс для информации о пользователе

```screenshots/``` — все скриншоты
