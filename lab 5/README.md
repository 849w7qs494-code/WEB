1. Научиться работать с базой данных MySQL через PHP
2. Создать таблицу для данных формы
3. Сохранять данные формы в базу данных

5. Использовать классы PHP для работы с таблицей
6. Работать с Docker контейнерами: nginx, PHP-FPM, MySQL, Adminer

## ⚙️ Как запустить проект

1. Клонировать репозиторий:
```bash
git clone https://github.com/849w7qs494-code/WEB.git
cd "WEB/lab 5"
```
2. Запустить контейнеры:
```bash
docker-compose up -d --build
```
3. Открыть в браузере:

```http://localhost:8080/form.html``` - форма

```http://localhost:8080/index.php``` - список студентов

```http://localhost:8081``` - adminer (сервер: db, пользователь: lab5_user, пароль: lab5_pass)

📂 Содержимое проекта

```www/form.html``` - форма регистрации студента

```www/index.php``` - список студентов
