## 📌 Описание задания

1. Настроить связку Nginx + PHP-FPM + MySQL в Docker
2. Создать таблицу students для хранения данных формы
3. Разработать класс Student для работы с таблицей (CRUD)
4. Сохранять данные из формы регистрации в базу данных
5. Выводить список студентов из БД на главной странице
6. Добавить возможность удаления записей
7. Подключить Adminer для управления базой данных

### Результат доступен по адресу:
**Форма регистрации:** http://localhost:8080/form.html

**Список студентов:** http://localhost:8080/index.php

**Adminer (управление БД):** http://localhost:8081

## ⚙️ Как запустить проект

1. Клонировать репозиторий:
```bash
git clone https://github.com/849w7qs494-code/WEB.git
cd "WEB/lab 5"
```
2. Запустить контейнеры:
```bash
docker-compose up -d --build
```
3. Открыть в браузере:

```http://localhost:8080/form.html``` - форма

```http://localhost:8080/index.php``` - список студентов

```http://localhost:8081``` - adminer (сервер: db, пользователь: lab5_user, пароль: lab5_pass)

📂 Содержимое проекта

```www/form.html``` - форма регистрации студента

```www/index.php``` - список студентов

```www/db.php``` - подключение к MySQL

```www/Student.php``` - класс для работы с таблицей
