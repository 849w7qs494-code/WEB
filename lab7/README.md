## 📌 Описание задания

1. Настроить Docker-окружение с Apache Kafka и Zookeeper
2. Реализовать класс QueueManager для работы с Kafka
3. Создать Producer (отправитель) для отправки сообщений
4. Создать Consumer (обработчик) для асинхронной обработки сообщений
5. Реализовать веб-интерфейс для отправки и просмотра сообщений
