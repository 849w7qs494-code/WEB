## 📌 Описание задания

1. Настроить Docker-окружение с PHP 8.2 + Apache для запуска приложения без локальной установки
2. Реализовать класс Student для регистрации студентов 
3. Написать unit-тесты для проверки валидации данных и успешной регистрации
4. Использовать mock-объект PDO для изоляции тестов от реальной базы данных
5. Протестировать HTTP-запрос через Guzzle с использованием MockHandler 
6. Настроить переменные окружения в файле .env.test и обеспечить изоляцию тестовой среды в Docker

## ⚙️ Как запустить проект

1. Клонировать репозиторий:
```bash
git clone https://github.com/849w7qs494-code/WEB.git
cd "WEB/lab8"
```
2. Запустить контейнеры:
```bash
docker-compose up -d --build
```
