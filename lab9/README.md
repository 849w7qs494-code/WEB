## 📌 Описание задания

1. Настроить CI/CD pipeline для автоматического тестирования PHP-приложения регистрации студентов
2. Создать Docker-образ на базе PHP 8.5-FPM с необходимыми расширениями (PDO, MySQL)
3. Реализовать класс `Student` с логикой валидации данных при регистрации
4. Написать unit-тесты с использованием PHPUnit для проверки корректности регистрации и обработки ошибок
5. Настроить конфигурацию `docker-compose.yml` для монтирования исходного кода и тестов в контейнер
6. Интегрировать GitHub Actions workflow для автоматического запуска сборки

## ⚙️ Как запустить проект

1. Клонировать репозиторий:
```bash
git clone https://github.com/849w7qs494-code/WEB.git
cd "WEB/lab9"
```
2. Запустить контейнеры:
```bash
docker-compose up -d --build
```
